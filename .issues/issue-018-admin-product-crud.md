# Issue 018: Admin Product Catalog CRUD (A-01)

## What to build
Develop the Admin Product Catalog CRUD dashboard enabling the administrator to create, read, update, active/deactive status, and delete products, including technical specifications per product category and image uploading (via Base64 compressed files) or URL inputs.

## Acceptance criteria
- [x] FrontEnd renders product table displaying thumbnail preview, name, category, price, stock, and status.
- [x] Admin can add/edit product name, category, price, and stock count.
- [x] Admin can configure category-specific specifications (cores, vram, tdp, wattage, etc.).
- [x] Admin can paste external image URLs or click **Upload Image** to compress local files to base64 WebP.
- [x] Admin can toggle is_active status of products to control catalog visibility.
- [x] All 95 tests pass successfully.
