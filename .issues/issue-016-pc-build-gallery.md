# Issue 016: SDLC Phase 4 (Development) - PC Build Gallery & Spec Cloning

## What to build
พัฒนาระบบเว็บบอร์ดแกลเลอรี่ประกอบคอมพิวเตอร์สาธารณะ (PC Build Gallery) เพื่อให้ลูกค้ากดแชร์สเปคเครื่องอ้างอิงของตนเองหลังประกอบเสร็จ และเปิดให้ลูกค้ารายอื่นสามารถดูสเปค กดถูกใจ และกดดึงข้อมูลสเปคนั้นกลับไปแก้ไข/สั่งประกอบต่อได้

## Acceptance criteria
- [ ] พัฒนาปุ่ม "แชร์สเปคคอมลงแกลเลอรี่ (Share Build)" ในหน้าคำสั่งซื้อของลูกค้า โดยสามารถป้อนชื่อหัวข้อ โพสต์รีวิว และอัปโหลดภาพเคสคอมพิวเตอร์ประกอบเสร็จ
- [ ] โพสต์ที่ถูกสร้างจะถูกจัดเก็บลงตาราง `gallery_posts` ในสถานะ `'Pending'` (รอการอนุมัติความเหมาะสมรูปภาพโดยผู้จัดการร้าน)
- [ ] พัฒนาหน้าจอเว็บบอร์ดแกลเลอรี่แสดงภาพสเปคเครื่องที่ผ่านอนุมัติแล้ว ลูกค้าท่านอื่นสามารถกดคลิกดูรายละเอียดสินค้าครบทั้ง 8 ชิ้นของโพสต์นั้น และมีปุ่มกดถูกใจ (Like) เพื่อเพิ่มยอดในคอลัมน์ `likes_count`
- [ ] พัฒนาระบบโคลนสเปคคอมฯ (Cloning specs logic): เมื่อผู้ใช้กดปุ่ม "แต่งต่อด้วยสเปคนี้ (Edit with this Build)" ระบบจะทำการคัดลอกรายการสินค้าทั้งหมดที่อยู่ในโพสต์นั้นเข้าไปเซ็ตเป็นค่าเริ่มต้นในหน้าจอ PC Builder ของลูกค้ารายใหม่ทันที
- [ ] มีตรรกะแจ้งเตือนตรวจสอบสินค้าเลิกจำหน่าย (Discontinued Check): หากมีสินค้าชิ้นใดชิ้นหนึ่งในโพสต์ถูก Disable (`is_active = false`) ให้เรนเดอร์ติดป้ายกำกับป้าย Badge สัญญาเตือนสีส้ม `[เลิกจำหน่าย]` และคัดลอกสินค้าชิ้นที่ยังขายได้เข้าไป พร้อมแจ้งเตือนให้ผู้ใช้งานเลือกฮาร์ดแวร์ชิ้นใหม่มาทดแทนอุปกรณ์ที่ขาดหายไปนั้น
