# Issue 008: SDLC Phase 3 (Design) - UML Sequence & Activity Diagrams

## What to build
จัดทำและออกแบบแผนภาพพฤติกรรมการทำงานของระบบยามรันไทม์ (UML Dynamic Diagrams) ประกอบด้วย Sequence Diagram และ Activity Diagram เพื่ออธิบายกระบวนการส่งต่อข้อมูลแบบเรียลไทม์

## Acceptance criteria
- [ ] แผนภาพ UML Sequence Diagram แสดงขั้นตอนจัดสเปคเครื่องและสั่งซื้อตั้งแต่ Customer เลือกสินค้า ➔ ตรวจสอบ Compatibility ➔ Checkout ➔ แนบหลักฐานสลิป ➔ อัปเดตยอดสั่งซื้อ สำเร็จเรียบร้อย
- [ ] แผนภาพ UML Activity Diagram แสดงกระบวนการทำงานของพนักงานประกอบเครื่อง (Staff) ตั้งแต่รับออเดอร์ ➔ ประกอบชิ้นส่วน ➔ ทดสอบ Burn-in Test (CPU/GPU Temp) ➔ อนุมัติผ่าน UAT ➔ กรอกเลข Tracking จัดส่งพัสดุ
- [ ] แผนภาพทั้งสองมีชื่อ Actor และส่วนเชื่อมต่อ API ตรงกับโครงสร้างข้อมูลใน Data Dictionary และ API Contract

## Blocked by
None - can start immediately
