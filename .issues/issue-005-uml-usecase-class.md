# Issue 005: SDLC Phase 2 (Analysis) - UML Use Case & Class Diagrams Design

## What to build
จัดทำและเขียนแผนภาพ UML Use Case Diagram และ UML Class Diagram เพื่อแสดงการเชื่อมโยงระบบและความสัมพันธ์ของออบเจ็กต์ต่างๆ ทางโครงสร้างซอฟต์แวร์

## Acceptance criteria
- [x] แผนภาพ UML Use Case Diagram แสดง Actors ทั้ง 4 และ Use Case ทั้งหมดถูกต้องครบถ้วนตามข้อกำหนดระบบ
- [x] แผนภาพ UML Class Diagram แสดงความสัมพันธ์ Multiplicity (1:1, 1:M, M:N) ระหว่างคลาสข้อมูลอย่างถูกต้อง
- [x] แนบแผนภาพ UML สำเร็จลงในเอกสาร SRS
