# Issue 001: SDLC Phase 1 (Planning) - Project Scope & Business Goals Analysis

## What to build
วิเคราะห์ขอบเขตโครงการและกำหนดเป้าหมายทางธุรกิจ (Business Goals) ของระบบ ComHub โดยสรุปผลลัพธ์ลงในเอกสารโครงการหลักเพื่อเป็นแนวทางเริ่มต้นตามกระบวนการ SDLC

## Acceptance criteria
- [ ] สรุปเป้าหมายธุรกิจอย่างชัดเจน 3 หัวข้อหลัก (ลดปัญหาคืนชิ้นส่วนไม่เข้ากัน, แนะนำกำลังวัตต์ PSU, ระบบติดตามประกอบแบบเรียลไทม์)
- [ ] กำหนดของเขตการทำงานแยกตาม Actor ทั้ง 4 บทบาท (Customer, Staff, Manager, Admin)
- [ ] มีการระบุ Main Functions สำคัญ 5 ส่วน (Catalog, PC Builder, Basket & Checkout, Order Management, Product Management)
- [ ] ผ่านการตรวจสอบและเห็นชอบจากสมาชิกและอาจารย์ผู้สอนวิชา csi 204

## Blocked by
None - can start immediately
