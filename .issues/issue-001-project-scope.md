# Issue 001: SDLC Phase 1 (Planning) - Project Scope & Business Goals Analysis

## What to build

วิเคราะห์ขอบเขตโครงการและกำหนดเป้าหมายทางธุรกิจ (Business Goals) ของระบบ ComHub โดยสรุปผลลัพธ์ลงในเอกสารโครงการหลักเพื่อเป็นแนวทางเริ่มต้นตามกระบวนการ SDLC

## Acceptance criteria

- [X] สรุปเป้าหมายธุรกิจอย่างชัดเจน 3 หัวข้อหลัก (ลดปัญหาคืนชิ้นส่วนไม่เข้ากัน, แนะนำกำลังวัตต์ PSU, ระบบติดตามประกอบแบบเรียลไทม์)
- [X] กำหนดของเขตการทำงานแยกตาม Actor ทั้ง 4 บทบาท (Customer, Staff, Manager, Admin)
- [X] มีการระบุ Main Functions สำคัญ 5 ส่วน (Catalog, PC Builder, Basket & Checkout, Order Management, Product Management)
- [X] ผ่านการตรวจสอบและเห็นชอบจากสมาชิกและอาจารย์ผู้สอนวิชา csi 204
